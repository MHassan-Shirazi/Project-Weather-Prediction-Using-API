# Project Weather Prediction Using API

## Overview
This project aims to provide accurate weather predictions using various APIs. The system fetches weather data and analyzes it to deliver forecasts based on user query parameters.

## Features
- Real-time weather data retrieval using APIs.
- User-friendly interface for displaying predictions.
- Support for multiple locations and weather types.

## Technologies Used
- **Languages:** Python, JavaScript
- **Frameworks:** Flask (for backend), React (for frontend)
- **APIs:** OpenWeatherMap API, WeatherAPI

## Installation
To set up the project locally, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/MHassan-Shirazi/Project-Weather-Prediction-Using-API.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Project-Weather-Prediction-Using-API
   ```
3. Install required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
1. Start the server:
   ```bash
   python app.py
   ```
2. Access the application via `http://localhost:5000`.
3. Use the interface to enter location details for weather predictions.

## API Documentation
Refer to the [OpenWeatherMap API Documentation](https://openweathermap.org/api) for detailed information on how to access and utilize the weather data.
